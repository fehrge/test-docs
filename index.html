    Sites

    addSite

    POST /sites

    Create a Site

    Body parameter

    {
      "required": [
        "server_id",
        "domain"
      ],
      "type": "object",
      "properties": {
        "id": {
          "type": "integer",
          "format": "int64",
          "readOnly": true
        },
        "server_id": {
          "type": "integer",
          "format": "int64",
          "example": 1
        },
        "domain": {
          "type": "string",
          "example": "hellfish.media"
        }
      }
    }

    Parameters

    Name In Type Required Description
    body body Site true Site object that needs to be created.
